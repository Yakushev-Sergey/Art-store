# Art-tore

**Живое демо:** https://yakushev-sergey.github.io/Art-tore/

## 📦 Функционал

- ✅ Адаптивная верстка (мобильные, планшеты, десктоп)
- ✅ Табы для фильтрации товаров
- ✅ Добавление товаров в корзину
- ✅ Счетчик количества товаров
- ✅ Автоматический подсчет общей стоимости
- ✅ Удаление товаров из корзины

## 🛠️ Технологии

| Технология | Что сделано |
|------------|--------------|
| **HTML5** | Семантическая верстка |
| **CSS3** | Flexbox, Grid, адаптив (mobile-first) |
| **JavaScript** | Табы, корзина, подсчет цены |
| **Gulp** | Сборка проекта |
